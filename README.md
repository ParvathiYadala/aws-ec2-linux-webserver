# AWS EC2 Linux Web Server Project

## Project Description
This project demonstrates hosting a static website on an AWS EC2 Ubuntu instance using Apache.

## Technologies Used
- AWS EC2
- Ubuntu Linux
- Apache Web Server
- SSH
- HTML

## Steps Performed
1. Created AWS account and launched EC2 instance
2. Connected to EC2 using SSH
3. Installed and managed Apache
4. Deployed static website using index.html
5. Checked logs and managed services
6. Stopped EC2 to avoid charges

## Outcome
Successfully hosted a static website on AWS EC2.
